# 《时空故事：青岚镇》MVP 完整故事线设计

## 1. 故事线总览

《青岚镇》MVP 由一条时间压力主线和四条交织的核心故事线构成。

```text
时间压力主线：30 天内，青岚镇从表面平静逐步走向危机。

核心故事线：
1. 疫病与药材线
2. 黑岭盗与商路线
3. 粮价与流民线
4. 旧庙火案真相线
```

四条线共享世界变量，并且互相影响：

```text
黑岭盗劫商路 → 药材和粮食减少 → 疫病与粮价恶化 → 镇民恐慌上升 → 秩序下降 → 巡守分身乏术 → 黑岭盗更容易行动。
```

玩家可以选择重点处理某条线，但不能完全无视其他线。未处理的危机会继续发展，并通过 NPC 行动推动故事。

## 2. 三十天基础时间线

| 时间 | 世界基础变化 | 主要触发线 |
|---|---|---|
| 第 1 天 | 玩家抵达青岚镇，镇中表面平静 | 入镇、认识 NPC |
| 第 2 天 | 镇门流民增多，集市粮价略涨 | 流民线、粮价线 |
| 第 3 天 | 第一批病人出现在药铺 | 疫病线 |
| 第 5 天 | 山路商队被劫，药材和粮食受影响 | 匪患线、商路线 |
| 第 7 天 | 集市出现米粮短缺传闻 | 粮价线 |
| 第 9 天 | 民居区病户增加，药铺开始吃紧 | 疫病线、人情线 |
| 第 10 天 | 沈青禾可能请求玩家协助药铺 | 疫病线 |
| 第 12 天 | 若药材不足，沈青禾可能独自上山 | 疫病线、山林线 |
| 第 14 天 | 陆怀山开始调查盗匪内应 | 匪患线、旧案线 |
| 第 15 天 | 赵三娘可出售关键情报 | 酒肆线 |
| 第 17 天 | 魏长庚开始压制旧案记录 | 旧案线 |
| 第 18 天 | 集市可能爆发流民冲突 | 粮价线、流民线 |
| 第 20 天 | 黑岭盗公开威胁青岚镇 | 匪患线 |
| 第 22 天 | 破驿旧庙可能被发现 | 旧案线、山林线 |
| 第 24 天 | 药铺、集市、镇门进入综合危机期 | 多线交织 |
| 第 26 天 | 最终危机开始酝酿 | 结局预备 |
| 第 28 天 | 各核心 NPC 根据状态做最后选择 | NPC 命运 |
| 第 30 天 | 结局判定 | 多结局 |

## 3. 主线一：疫病与药材线

### 3.1 核心人物

- 沈青禾
- 老药师
- 小宛
- 病户家属

### 3.2 核心地点

- 回春药铺
- 民居区
- 青岚山林
- 破驿旧庙

### 3.3 起点

第 3 天，第一批发热咳嗽的病人来到回春药铺。沈青禾注意到这些人都曾接触过山林方向的水源、草药或外来货物，但她缺少证据。

### 3.4 阶段推进

#### 阶段一：病人出现

触发条件：

```yaml
trigger:
  all:
    - fact: time.day
      greater_than_or_equal: 3
    - fact: facts.first_patient_event_happened
      not_equals: true
```

结果：

```text
plague_level +10
herb_stock -5
药铺 patient_count +3
沈青禾 mood = anxious
```

玩家可选：

| 选项 | 条件 | 后果 |
|---|---|---|
| 帮忙诊治 | 医者身份或 medicine 足够 | plague_level -5，沈青禾 trust +10 |
| 出钱买药 | money 足够 | plague_level -3，gratitude +10 |
| 旁观 | 无 | 疫病继续自然增长 |

#### 阶段二：药材告急

当 `plague_level >= 35` 或 `herb_stock <= 40` 时触发。

结果：

```text
药铺交互增加“帮忙熬药”“调查药材来源”。
沈青禾开始向信任玩家求助。
民居区 infected_households 增加。
```

玩家可选：

| 选项 | 后果 |
|---|---|
| 在药铺帮忙 | 疫病小幅下降，消耗药材和体力 |
| 去山林采药 | 获得药材，但可能遇到盗匪 |
| 调查药材来源 | 可能发现商路遇袭和旧庙草药残页 |
| 买药囤积 | 对自身有利，但会加剧药铺短缺 |

#### 阶段三：沈青禾上山

如果满足：

```yaml
conditions:
  all:
    - fact: variables.plague_level
      greater_than_or_equal: 45
    - fact: variables.herb_stock
      less_than: 25
    - fact: facts.shen_collected_herbs
      not_equals: true
```

沈青禾会自行前往青岚山林。

可能结果：

| 玩家行为 | 结果 |
|---|---|
| 及时同行或寻找 | 沈青禾安全返回，herb_stock +20 |
| 武人护送 | 降低盗匪威胁，陆怀山 trust +5 |
| 医者识别药材 | plague_level 大幅下降 |
| 忽视 | 沈青禾可能受伤、失踪或被盗匪抓走 |

#### 阶段四：疫病真因

当玩家通过药铺后院、山林草药坡或旧庙残页获得足够线索后，可发现：疫病并非单纯天灾，而与山林中被污染的药草、水源及旧庙暗室有关。

关键 facts：

```yaml
facts:
  shen_hidden_herb_note_found: true
  forest_sick_herb_found: true
  old_temple_cellar_found: true
```

当上述线索满足至少两项，解锁“疫病真因”事件。

### 3.5 结局影响

| 状态 | 影响 |
|---|---|
| plague_level <= 40 | 有利于“青岚得存” |
| plague_level >= 75 | 进入“空镇余烬”高优先级判断 |
| 沈青禾存活且信任高 | 可触发药铺救治总结 |
| 沈青禾失踪或死亡 | 药铺线悲剧，疫病更难压制 |

## 4. 主线二：黑岭盗与商路线

### 4.1 核心人物

- 陆怀山
- 韩烬
- 赵三娘
- 商队管事
- 盗匪内应

### 4.2 核心地点

- 镇门
- 青岚山林
- 旧酒肆
- 破驿旧庙

### 4.3 起点

第 5 天，一支商队在山路上被劫，药材和粮食损失严重。镇门开始流传黑岭盗重新聚集的消息。

### 4.4 阶段推进

#### 阶段一：商路遇袭

效果：

```text
merchant_route_safety -20
herb_stock -10
food_price +15
town_fear +10
bandit_power +5
```

玩家可选：

| 选项 | 后果 |
|---|---|
| 协助陆怀山追查 | 解锁盗匪踪迹，陆怀山 trust +10 |
| 出资补贴商队 | 商路安全小幅回升，行商身份收益更高 |
| 置之不理 | bandit_power +5，后续夜袭风险上升 |

#### 阶段二：内应疑云

当 `bandit_power >= 55` 或玩家获得酒肆情报后，陆怀山开始怀疑镇中有人与黑岭盗接触。

可能嫌疑方向：

- 米铺掌柜罗守仓：为了保护粮仓，曾与可疑人物接触。
- 酒肆后堂客人：可能传递消息。
- 镇长旧部：与旧庙火案有关。

这条线不应设计成单一答案，而是多条线索指向“黑岭盗利用镇中旧案与利益关系渗透”。

#### 阶段三：山林追踪

玩家可在山林追踪盗匪。

| 身份 | 特殊选项 |
|---|---|
| 落魄武人 | 正面追击、埋伏盗匪 |
| 外乡行商 | 利用商队路线设局 |
| 游方医者 | 借寻找草药发现盗匪临时营地 |

追踪成功后可发现：盗匪并不只是抢粮抢药，还在寻找旧庙暗室里的东西。

#### 阶段四：韩烬逼近

当：

```yaml
conditions:
  all:
    - fact: variables.bandit_power
      greater_than_or_equal: 80
    - fact: variables.guard_power
      less_than_or_equal: 40
```

触发“黑岭盗夜袭”。

若玩家提前降低 bandit_power、增强 guard_power 或揭露内应，则夜袭可能被推迟、削弱或转化为山林决战。

### 4.5 结局影响

| 状态 | 影响 |
|---|---|
| bandit_power <= 50 | 有利于“青岚得存” |
| bandit_power >= 80 且 guard_power <= 40 | 触发“匪火入镇” |
| 韩烬死亡或被击退 | 商路恢复，guard_power 上升 |
| 玩家与韩烬合作 | 可能进入“暗约成局” |

## 5. 主线三：粮价与流民线

### 5.1 核心人物

- 罗守仓
- 小宛
- 赵三娘
- 流民群体

### 5.2 核心地点

- 集市
- 镇门
- 民居区
- 旧酒肆

### 5.3 起点

第 2 天起，镇门外流民增加。第 5 天商路遇袭后，粮价上涨。第 7 天，集市出现米粮短缺传闻。

### 5.4 阶段推进

#### 阶段一：粮价上涨

```text
food_price +15
refugee_pressure +10
market.conflict_level +5
```

玩家可通过购买、捐助、谈判、调查来影响粮价线。

#### 阶段二：罗守仓囤粮

当 `food_price >= 130` 且 `town_fear >= 40` 时，罗守仓可能开始暗中囤粮。

罗守仓不是单纯反派。他囤粮的原因有三层：

1. 想趁乱获利。
2. 害怕流民冲击米铺。
3. 曾经历饥荒，因此对失去粮食有极强恐惧。

玩家可选：

| 选项 | 后果 |
|---|---|
| 揭露囤粮 | food_price 降低，但集市短期混乱 |
| 逼迫开仓 | 快速缓解流民压力，但罗守仓敌对 |
| 说服平价放粮 | 需要高关系或谈判，效果最好 |
| 默许囤粮 | 罗守仓可能帮助玩家，但流民压力升高 |

#### 阶段三：流民冲突

如果 `refugee_pressure >= 50` 且 `food_price >= 150`，集市或镇门触发流民冲突。

玩家可选：

| 选项 | 后果 |
|---|---|
| 安抚流民 | reputation +，town_fear - |
| 驱逐流民 | town_order 短期 +，refugee_pressure 转为仇恨 |
| 分粮救助 | food_price 降低，money 或 food 消耗 |
| 借巡守镇压 | guard_power 消耗，陆怀山关系视选择变化 |

#### 阶段四：小宛命运

小宛作为流民代表，会根据玩家行为走向不同命运：

- 被救助：成为民居区线索来源，结局中提及玩家仁名。
- 被驱逐：可能在山林失踪，增加镇民恐慌。
- 被利用：可能成为盗匪或赵三娘情报线的一环。

### 5.5 结局影响

| 状态 | 影响 |
|---|---|
| food_price <= 130 | 有利于小镇稳定 |
| refugee_pressure <= 40 | 降低暴动风险 |
| 小宛存活且获救 | 有利于“仁名远播”类评价 |
| 罗守仓被杀或米铺被抢 | 短期粮价下降，长期 town_order 大幅下降 |

## 6. 主线四：旧庙火案真相线

### 6.1 核心人物

- 魏长庚
- 陆怀山
- 赵三娘
- 许闻舟
- 韩烬
- 老祠祝

### 6.2 核心地点

- 祠堂
- 旧酒肆
- 青岚山林
- 破驿旧庙

### 6.3 起点

玩家初期只能听到模糊传闻：多年前破驿旧庙曾有一场火，死过人，后来镇长压下此事。

旧案线是隐藏线，不会一开始强行暴露。

### 6.4 线索来源

| 来源 | 线索 |
|---|---|
| 祠堂旧档 | 当年旧庙火案记录缺页 |
| 赵三娘情报 | 当年有人从旧庙救出过一个孩子 |
| 陆怀山调查 | 镇长阻止巡守查旧案 |
| 沈青禾药草残页 | 旧庙附近有特殊草药和暗室 |
| 韩烬踪迹 | 黑岭盗也在寻找旧庙暗室 |
| 许闻舟手稿 | 旧案可能与他父亲有关 |

### 6.5 阶段推进

#### 阶段一：旧案传闻

在酒肆或祠堂可打听到。

效果：

```text
truth_progress +5
解锁许闻舟或老祠祝对话
```

#### 阶段二：旧档房

玩家需要解除祠堂档房锁定。

方式：

- 获得魏长庚许可。
- 说服老祠祝。
- 夜探祠堂。
- 与陆怀山合作调查。

效果：

```text
truth_progress +15
old_case_record_found = true
```

#### 阶段三：魏长庚压制真相

当 `truth_progress >= 40` 时，魏长庚触发压制行为。

可能结果：

| 玩家选择 | 后果 |
|---|---|
| 正面质问 | 魏长庚警觉，部分线索被封锁 |
| 暗中调查 | 可继续推进，但风险增加 |
| 借陆怀山施压 | town_order 可能下降，但真相推进 |
| 暂时隐忍 | 错过部分时间，但保留魏长庚关系 |

#### 阶段四：发现旧庙

当满足以下任一条件，旧庙解锁：

```yaml
any:
  - old_case_record_found = true
  - shen_hidden_herb_note_found = true
  - player_tracked_bandit_to_old_temple = true
  - zhao_sold_old_temple_clue = true
```

#### 阶段五：旧庙暗室

旧庙暗室中可以发现最终证据：

- 当年火案并非意外。
- 魏长庚隐瞒证词是为了保护镇中某些人，也为了保住自己的权力。
- 韩烬知道旧庙暗室，因为他曾与当年幸存者有关。
- 疫病线的特殊草药、水源与旧庙附近环境有关。

### 6.6 结局影响

| 状态 | 影响 |
|---|---|
| truth_progress >= 80 | 可进入“真相昭雪” |
| truth_progress 高但 town_order 低 | 真相揭开但小镇动荡 |
| 魏长庚被提前逼死或杀死 | 真相可能残缺 |
| 许闻舟存活 | 真相结局文本更完整 |

## 7. NPC 自主行动总表

| NPC | 触发条件 | 自主行动 | 影响 |
|---|---|---|---|
| 沈青禾 | plague_level 高且 herb_stock 低 | 上山采药 | 可能救药铺，也可能失踪 |
| 陆怀山 | bandit_power 高 | 调查盗匪内应 | 推进匪患线，可能触及旧案 |
| 赵三娘 | town_fear 高 | 散布或售卖传闻 | 影响玩家线索获取 |
| 魏长庚 | truth_progress 高 | 封锁旧档 | 延缓真相线 |
| 韩烬 | bandit_power 高且 guard_power 低 | 夜袭青岚镇 | 可能进入匪火结局 |
| 罗守仓 | food_price 高且 town_fear 高 | 囤粮 | 粮价上升，集市冲突增加 |
| 小宛 | refugee_pressure 高 | 求助、偷粮或逃离 | 影响流民线和人情评价 |
| 许闻舟 | truth_progress 中等 | 深入调查旧案 | 可能发现旧庙线索，也可能被威胁 |

## 8. 玩家主线介入方式

### 8.1 温和救助路线

特点：

- 帮药铺。
- 救助流民。
- 安抚集市。
- 与陆怀山合作。
- 尽量保住 NPC。

可能结果：

- 青岚得存。
- 仁名远播。
- 真相部分揭开。

### 8.2 强硬压制路线

特点：

- 用武力处理盗匪和冲突。
- 威胁罗守仓开仓。
- 夜探、逼问、强行调查。

可能结果：

- 匪患被压下。
- town_order 短期维持或崩坏。
- NPC 关系两极化。
- 可能进入血债类评价。

### 8.3 利益交易路线

特点：

- 与赵三娘、罗守仓、商队合作。
- 用金钱购买情报和资源。
- 可能与韩烬或魏长庚达成暗约。

可能结果：

- 暗约成局。
- 小镇表面稳定但真相被压下。
- 玩家获得利益，但部分 NPC 命运恶化。

### 8.4 真相优先路线

特点：

- 集中调查祠堂、酒肆、旧庙。
- 与陆怀山、许闻舟合作。
- 不一定能兼顾疫病与粮价。

可能结果：

- 真相昭雪。
- 如果其他变量恶化，可能出现“真相揭开但小镇已残”的变体。

## 9. 关键因果链示例

### 9.1 药铺崩溃链

```text
玩家忽视第一批病人
→ plague_level 持续上升
→ herb_stock 降低
→ 沈青禾独自上山
→ 玩家未及时寻找
→ 沈青禾失踪
→ 药铺治疗能力下降
→ 疫病失控
→ 空镇余烬
```

### 9.2 匪患失控链

```text
商路遇袭后玩家未协助追查
→ bandit_power 上升
→ 陆怀山调查受阻
→ guard_power 被集市冲突消耗
→ 韩烬发动夜袭
→ 匪火入镇
```

### 9.3 真相昭雪链

```text
玩家在酒肆获得旧案传闻
→ 说服老祠祝打开旧档房
→ truth_progress 上升
→ 魏长庚试图封锁记录
→ 玩家与陆怀山合作继续调查
→ 发现破驿旧庙
→ 找到暗室证据
→ 真相昭雪
```

### 9.4 粮价暴乱链

```text
商路遇袭
→ food_price 上升
→ 罗守仓囤粮
→ 玩家未处理流民压力
→ 集市冲突升级
→ 米铺被抢或流民被驱逐
→ town_order 下降
→ 其他危机更难处理
```

## 10. 结局判定逻辑

第 30 天夜晚结束后，系统按优先级检查结局。

优先级建议：

```text
特殊结局 > 灾难结局 > 真相结局 > 稳定结局 > 普通离开结局
```

例如：

1. 如果 bandit_power 极高且 guard_power 极低，优先进入“匪火入镇”。
2. 如果 plague_level 极高且 town_fear 极高，进入“空镇余烬”。
3. 如果 truth_progress 足够高且关键 NPC 存活，进入“真相昭雪”。
4. 如果多项危机都被控制，进入“青岚得存”。
5. 如果玩家存活但小镇状态普通或失败，进入“孤身远行”。

## 11. 故事线设计原则

1. 每条故事线都有 NPC 自主推动，不等待玩家。
2. 玩家可以延缓、改变或利用故事线，但不能让世界静止。
3. 每条线至少有一名核心 NPC 和一个核心地点。
4. 每条线都要影响至少两个世界变量。
5. 每条线都要能贡献到至少一个结局。
6. 所有重大事件都要生成因果链日志。
